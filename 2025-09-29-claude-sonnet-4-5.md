# Introducing Claude Sonnet 4.5

> 原文链接: https://www.anthropic.com/news/claude-sonnet-4-5
> 日期: 2025-09-29

---

## English

Anthropic has released Claude Sonnet 4.5, described as "the best coding model in the world" with capabilities for building complex agents and computer use tasks. The model demonstrates substantial improvements in reasoning, mathematics, and domain-specific knowledge across finance, law, medicine, and STEM fields.

### Key Features

**Coding Performance:** Claude Sonnet 4.5 achieves state-of-the-art results on SWE-bench Verified and can maintain focus for more than 30 hours on complex, multi-step tasks.

**Computer Use:** The model leads on OSWorld benchmarks at 61.4%, up from Sonnet 4's 42.2% four months prior. It powers the Claude for Chrome extension for real-world browser automation.

**Pricing:** Remains unchanged from Claude Sonnet 4 at $3/$15 per million tokens, making it a drop-in replacement.

### Product Updates

- Claude Code now includes checkpoints for progress saving and rollback features
- Native VS Code extension available
- Context editing and memory tools added to Claude API
- File creation capabilities (spreadsheets, slides, documents) in Claude apps
- Claude Agent SDK released for developers to build custom agents

### Safety and Alignment

This is Anthropic's most aligned frontier model yet, with reductions in sycophancy, deception, and power-seeking behaviors. Released under AI Safety Level 3 protections with CBRN classifiers, though false positives have been reduced tenfold since initial implementation.

### Customer Testimonials

Partners including Cursor, GitHub Copilot, Netflix, Canva, and Figma report significant improvements in code quality, planning performance, and development velocity with the new model.

### Availability

Claude Sonnet 4.5 is available immediately via the Claude API, Claude apps, and Claude Code across all platforms.

## 中文翻译

Anthropic 发布了 Claude Sonnet 4.5，被描述为"世界上最好的编程模型"，具有构建复杂代理和计算机使用任务的能力。该模型在推理、数学以及金融、法律、医学和 STEM 领域的特定知识方面展示了显著改进。

### 关键特性

**编程性能：** Claude Sonnet 4.5 在 SWE-bench Verified 上取得了最先进的结果，能够在复杂的多步骤任务上保持超过 30 小时的专注。

**计算机使用：** 该模型在 OSWorld 基准测试中以 61.4% 领先，较四个月前 Sonnet 4 的 42.2% 有所提升。它为 Claude for Chrome 扩展提供支持，实现真实世界的浏览器自动化。

**定价：** 与 Claude Sonnet 4 保持不变，为每百万 token 3 美元/15 美元，使其成为即插即用的替代品。

### 产品更新

- Claude Code 现在包含检查点，支持进度保存和回滚功能
- 提供原生 VS Code 扩展
- Claude API 新增上下文编辑和记忆工具
- Claude 应用中新增文件创建功能（电子表格、幻灯片、文档）
- 发布 Claude Agent SDK，供开发者构建自定义代理

### 安全和对齐

这是 Anthropic 迄今为止对齐度最高的前沿模型，在谄媚、欺骗和权力寻求行为方面有所减少。在 AI 安全级别 3 保护下发布，配备 CBRN 分类器，但误报率已比初始实现减少了十倍。

### 客户评价

包括 Cursor、GitHub Copilot、Netflix、Canva 和 Figma 在内的合作伙伴报告了代码质量、规划性能和开发速度的显著改善。

### 可用性

Claude Sonnet 4.5 可通过 Claude API、Claude 应用和 Claude Code 在所有平台上立即使用。
