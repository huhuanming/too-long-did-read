## English

Anthropic unveiled Claude Opus 4.5, their newest AI model, making it immediately available across their applications, API, and major cloud platforms. The model represents advancement in artificial intelligence capabilities, particularly excelling at coding, agent development, and computer use tasks.

### Key Capabilities

The model demonstrates state-of-the-art performance on software engineering benchmarks. "Claude Opus 4.5 is state-of-the-art on tests of real-world software engineering." Developers can access it via the Claude API using the model identifier `claude-opus-4-5-20251101` at pricing of $5/$25 per million tokens.

Improvements span multiple domains including vision, reasoning, and mathematics. The model achieved notably superior results across programming languages and complex problem-solving scenarios.

### Developer Platform Enhancements

New features include an effort parameter allowing developers to balance computation time against capability requirements. "Set to a medium effort level, Opus 4.5 matches Sonnet 4.5's best score on SWE-bench Verified, but uses 76% fewer output tokens."

Additional tools support longer-running agents and expanded integration with Excel, Chrome, and desktop environments. Claude Code received upgrades enabling more precise planning and parallel session management.

### Safety Advances

Anthropic characterizes Opus 4.5 as their "most robustly aligned model" released to date. The model demonstrates enhanced resistance to prompt injection attacks compared to competing frontier models, addressing security concerns for production deployments.

### Customer Feedback

Technology leaders from multiple organizations provided testimonials highlighting token efficiency improvements, superior agentic task performance, and code generation capabilities exceeding predecessor models.

## 中文翻译

Anthropic 发布了 Claude Opus 4.5，使其最新 AI 模型立即在其应用程序、API 和主要云平台上可用。该模型代表了人工智能能力的进步，特别在编程、代理开发和计算机使用任务方面表现出色。

### 关键能力

该模型在软件工程基准测试上展示了最先进的性能。"Claude Opus 4.5 在真实世界软件工程测试中达到最先进水平。"开发者可以通过 Claude API 使用模型标识符 `claude-opus-4-5-20251101` 访问，定价为每百万 token 5 美元/25 美元。

改进涵盖多个领域，包括视觉、推理和数学。该模型在编程语言和复杂问题解决场景中取得了显著优越的结果。

### 开发者平台增强

新功能包括一个 effort 参数，允许开发者在计算时间和能力需求之间取得平衡。"设置为中等努力级别时，Opus 4.5 在 SWE-bench Verified 上匹配 Sonnet 4.5 的最佳得分，但使用的输出 token 减少 76%。"

额外的工具支持更长时间运行的代理和与 Excel、Chrome 和桌面环境的扩展集成。Claude Code 获得了升级，实现了更精确的规划和并行会话管理。

### 安全进步

Anthropic 将 Opus 4.5 描述为迄今发布的"最稳健对齐的模型"。与竞争的前沿模型相比，该模型展示了增强的提示注入攻击抵抗力，解决了生产部署的安全问题。

### 客户反馈

多个组织的技术领导者提供了评价，强调了 token 效率改进、卓越的智能体任务性能和超越前代模型的代码生成能力。
