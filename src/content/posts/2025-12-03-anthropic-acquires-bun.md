## English

Claude represents the world's most capable AI model for developers and enterprises. Claude Code, launched in May 2025, has achieved remarkable growth—reaching $1 billion in run-rate revenue within six months of public availability. Anthropic announced the acquisition of Bun, a JavaScript runtime platform founded by Jarred Sumner in 2021.

Bun functions as an all-in-one toolkit combining a runtime, package manager, bundler, and test runner. The platform significantly outperforms competing solutions in speed and reliability. This acquisition will enhance Claude Code's performance, stability, and capabilities for software development teams.

Since launch, Claude Code has become integral to major enterprises including Netflix, Spotify, KPMG, L'Oreal, and Salesforce. Bun has been instrumental in supporting Claude Code's infrastructure scaling throughout this expansion period.

Mike Krieger, Chief Product Officer at Anthropic, stated: "Bun represents exactly the kind of technical excellence we want to bring into Anthropic." The acquisition aligns with Anthropic's strategic approach to acquisitions that strengthen technical capabilities and enterprise AI leadership while maintaining alignment with company principles.

Bun currently receives over 7 million monthly downloads and has earned 82,000+ GitHub stars. Companies like Midjourney and Lovable have adopted Bun to improve development speed and productivity.

The platform will remain open-source and MIT-licensed, with Anthropic committed to advancing it as the preferred JavaScript and TypeScript development tool.

## 中文翻译

Claude 代表了面向开发者和企业的全球最强大的 AI 模型。Claude Code 于 2025 年 5 月推出，实现了惊人的增长——在公开可用后六个月内达到 10 亿美元的运营率收入。Anthropic 宣布收购 Bun——一个由 Jarred Sumner 于 2021 年创立的 JavaScript 运行时平台。

Bun 是一个集运行时、包管理器、打包器和测试运行器于一体的全能工具包。该平台在速度和可靠性方面显著优于竞争解决方案。此次收购将增强 Claude Code 对软件开发团队的性能、稳定性和能力。

自推出以来，Claude Code 已成为 Netflix、Spotify、KPMG、L'Oreal 和 Salesforce 等主要企业不可或缺的工具。Bun 在整个扩展期间一直在支持 Claude Code 的基础设施扩展方面发挥着重要作用。

Anthropic 首席产品官 Mike Krieger 表示："Bun 正是我们想要带入 Anthropic 的那种技术卓越。"此次收购符合 Anthropic 的战略方针——通过收购加强技术能力和企业 AI 领导地位，同时保持与公司原则的一致。

Bun 目前每月下载量超过 700 万，GitHub star 数超过 82,000。Midjourney 和 Lovable 等公司已采用 Bun 来提高开发速度和生产力。

该平台将继续保持开源和 MIT 许可，Anthropic 承诺将其推进为首选的 JavaScript 和 TypeScript 开发工具。
