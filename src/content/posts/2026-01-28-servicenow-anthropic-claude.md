## English

As enterprises transition from AI experimentation to production deployment across core operations, both scale and security prove equally important as raw capabilities.

ServiceNow, which manages and automates processes from IT support to HR to customer service on a unified platform, has selected Claude as its default model for ServiceNow Build Agent and as a preferred model throughout the ServiceNow AI Platform. Build Agent enables developers of all skill levels to construct and deploy Claude as an autonomous "agent" capable of reasoning, deciding on actions, and executing tasks independently.

ServiceNow is also distributing Claude and Claude Code across its global workforce exceeding 29,000 employees to streamline sales preparation. Early testing demonstrates a 95% reduction in seller preparation time, while Claude Code boosts engineering productivity by reducing the interval from concept to implementation.

ServiceNow's platform processes over 80 billion workflows annually—spanning security incident resolution, employee onboarding, and customer support queue management. Now these workflows gain access to Claude's reasoning and coding capabilities, complete with enterprise-grade access controls, usage monitoring, and compliance features.

"A common error enterprises make with AI is treating it as a 'bolt on' tool," stated Dario Amodei, CEO and co-founder of Anthropic. "The way to get much better results is making AI integral to how work gets done—woven into everything workers do daily."

Bill McDermott, ServiceNow chairman and CEO, remarked: "ServiceNow with Anthropic is turning intelligence into action through AI-native workflows for the world's largest enterprises."

### Claude for ServiceNow Customers

**Powering Enterprise App Development:** Claude serves as the default model for ServiceNow Build Agent, an enterprise-grade coding solution. ServiceNow anticipates usage will quadruple within twelve months.

**Accelerating Product Adoption and Time to Value:** ServiceNow targets a 50% reduction in customer implementation time, shortening the interval from initial sales discussions to autonomous deployment.

**Applying Innovative Claude-Powered Solutions to Industries:** ServiceNow is developing agentic applications for healthcare and life sciences, supporting research analysis, claims authorization, and more within ServiceNow's governed platform.

### How ServiceNow Will Use Claude Internally

**Transforming Sales Productivity:** ServiceNow sellers employ a Claude-powered coaching tool connecting Claude to real-time enterprise data and web search. Testing shows up to 95% reduction in preparation time.

**Boosting Productivity with Claude Code:** ServiceNow deployed Claude Code to engineers, developers, and technical teams for writing, reviewing, and debugging code, automating repetitive tasks, and accelerating internal tooling development.

### Availability

Claude is now available as the default Build Agent model and preferred model across the ServiceNow AI Platform. Tens of thousands of ServiceNow enterprise customers and the company's global workforce can access Claude.

ServiceNow 在统一平台上管理和自动化从 IT 支持到人力资源到客户服务的流程，已选择 Claude 作为 ServiceNow Build Agent 的默认模型和整个 ServiceNow AI 平台的首选模型。Build Agent 使各种技能水平的开发者能够构建和部署 Claude 作为能够推理、决定行动和独立执行任务的自主"代理"。

ServiceNow 还在其超过 29,000 名员工的全球团队中分发 Claude 和 Claude Code，以简化销售准备。早期测试显示卖方准备时间减少了 95%，而 Claude Code 通过缩短从概念到实施的时间来提高工程生产力。

ServiceNow 的平台每年处理超过 800 亿个工作流——涵盖安全事件解决、员工入职和客户支持队列管理。现在这些工作流可以利用 Claude 的推理和编程能力，配备企业级访问控制、使用监控和合规功能。

Anthropic CEO 兼联合创始人 Dario Amodei 表示："企业在 AI 方面常犯的一个错误是将其视为'附加'工具。获得更好结果的方法是使 AI 成为工作方式的组成部分——融入员工每天所做的一切中。"

ServiceNow 主席兼 CEO Bill McDermott 评论道："ServiceNow 与 Anthropic 正在通过面向世界最大企业的 AI 原生工作流将智能转化为行动。"

### Claude 为 ServiceNow 客户服务

**推动企业应用开发：** Claude 作为 ServiceNow Build Agent 的默认模型，这是一个企业级编程解决方案。ServiceNow 预计使用量将在十二个月内增长四倍。

**加速产品采用和价值实现：** ServiceNow 目标是将客户实施时间缩短 50%。

**转变销售生产力：** ServiceNow 销售人员使用 Claude 驱动的辅导工具，将 Claude 连接到实时企业数据和网络搜索。测试显示准备时间减少高达 95%。

**使用 Claude Code 提升生产力：** ServiceNow 向工程师、开发者和技术团队部署了 Claude Code，用于编写、审查和调试代码，自动化重复任务，并加速内部工具开发。
