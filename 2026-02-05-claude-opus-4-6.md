# Introducing Claude Opus 4.6

> 原文链接: https://www.anthropic.com/news/claude-opus-4-6
> 日期: 2026-02-05

---

## English

Anthropic announced Claude Opus 4.6, describing it as an upgrade to their most advanced model with enhanced coding capabilities. The new version "plans more carefully, sustains agentic tasks for longer" and features a 1M token context window in beta.

The model demonstrates state-of-the-art performance across multiple evaluations. It achieves the highest score on Terminal-Bench 2.0 for agentic coding and leads on Humanity's Last Exam, a multidisciplinary reasoning test. On GDPval-AA, an evaluation of economically valuable knowledge work, Opus 4.6 outperforms GPT-5.2 by approximately 144 Elo points and its predecessor by 190 points.

Key improvements include better long-context retrieval, where "on the 8-needle 1M variant of MRCR v2...Opus 4.6 scores 76%, whereas Sonnet 4.5 scores just 18.5%." The model excels at handling large codebases, identifying vulnerabilities, and performs nearly twice as well as Opus 4.5 on computational biology tests.

Product updates include agent teams in Claude Code, context compaction for longer tasks, adaptive thinking that automatically determines when extended reasoning helps, and effort controls (low, medium, high, max) for developers to balance intelligence, speed, and cost.

Claude in Excel received substantial upgrades, and Claude in PowerPoint launched in research preview. Both applications now handle complex tasks more effectively for everyday work scenarios.

Safety testing was extensive, with Opus 4.6 showing "low rates of misaligned behaviors" comparable to or better than other frontier models. The model demonstrates the lowest over-refusal rate among recent Claude versions.

Pricing remains at $5/$25 per million tokens for standard usage, with premium pricing ($10/$37.50) applying for prompts exceeding 200k tokens on the Claude Developer Platform.

## 中文翻译

Anthropic 宣布推出 Claude Opus 4.6，将其描述为对最先进模型的升级，具有增强的编程能力。新版本"规划更周密，维持智能体任务时间更长"，并具有测试阶段的 100 万 token 上下文窗口。

该模型在多项评估中展示了最先进的性能。它在智能体编程的 Terminal-Bench 2.0 上获得最高分，并在多学科推理测试 Humanity's Last Exam 上领先。在衡量经济价值知识工作的 GDPval-AA 评估中，Opus 4.6 以约 144 Elo 分的优势超过 GPT-5.2，以 190 分的优势超过其前代。

关键改进包括更好的长上下文检索能力——"在 MRCR v2 的 8 针 1M 变体上……Opus 4.6 得分 76%，而 Sonnet 4.5 仅得分 18.5%。"该模型擅长处理大型代码库、识别漏洞，在计算生物学测试中的表现几乎是 Opus 4.5 的两倍。

产品更新包括 Claude Code 中的智能体团队、用于更长任务的上下文压缩、自动确定何时扩展推理有帮助的自适应思维，以及用于开发者平衡智能、速度和成本的努力控制（低、中、高、最大）。

Claude in Excel 获得了大幅升级，Claude in PowerPoint 以研究预览形式推出。两个应用程序现在都能更有效地处理日常工作场景中的复杂任务。

安全测试非常广泛，Opus 4.6 显示出"低比率的错位行为"，与其他前沿模型相当或更好。该模型展示了最近 Claude 版本中最低的过度拒绝率。

标准用法定价保持在每百万 token 5 美元/25 美元，在 Claude 开发者平台上超过 20 万 token 的提示适用高级定价（10 美元/37.50 美元）。
