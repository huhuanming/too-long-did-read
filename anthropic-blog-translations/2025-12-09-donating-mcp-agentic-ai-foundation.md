# Donating the Model Context Protocol and Establishing the Agentic AI Foundation
# 捐赠模型上下文协议并成立代理式 AI 基金会

> Source: https://www.anthropic.com/news/donating-the-model-context-protocol-and-establishing-of-the-agentic-ai-foundation
> Date: 2025-12-09

---

Anthropic announced the donation of the Model Context Protocol (MCP) to the Linux Foundation's newly established Agentic AI Foundation (AAIF). This initiative represents a significant step toward ensuring MCP remains "open-source, community-driven and vendor-neutral."

Anthropic 宣布将模型上下文协议（MCP）捐赠给 Linux 基金会新成立的代理式 AI 基金会（AAIF）。这一举措代表着确保 MCP 保持"开源、社区驱动和厂商中立"的重要一步。

---

## Key Achievements of MCP
## MCP 的主要成就

Since its introduction one year ago, MCP has experienced substantial growth:

自一年前推出以来，MCP 经历了大幅增长：

- **Ecosystem expansion**: Over 10,000 active public MCP servers now exist, supporting developer tools and enterprise deployments
- **生态系统扩展：** 现在有超过 10,000 个活跃的公共 MCP 服务器，支持开发者工具和企业部署

- **Platform adoption**: Integration with ChatGPT, Cursor, Gemini, Microsoft Copilot, and Visual Studio Code
- **平台采用：** 与 ChatGPT、Cursor、Gemini、Microsoft Copilot 和 Visual Studio Code 集成

- **Infrastructure support**: Enterprise deployment capabilities from AWS, Cloudflare, Google Cloud, and Microsoft Azure
- **基础设施支持：** 来自 AWS、Cloudflare、Google Cloud 和 Microsoft Azure 的企业部署能力

## Recent Developments
## 近期进展

Anthropic continues expanding MCP's capabilities. Claude now features over 75 connectors powered by MCP. The platform recently introduced Tool Search and Programmatic Tool Calling features to optimize production deployments handling thousands of tools efficiently.

Anthropic 继续扩展 MCP 的能力。Claude 现在拥有超过 75 个由 MCP 驱动的连接器。该平台最近推出了工具搜索和程序化工具调用功能，以优化高效处理数千个工具的生产部署。

The November 25th specification release added asynchronous operations, statelessness, server identity, and official extensions. MCP SDKs for major programming languages have achieved 97 million+ monthly downloads across Python and TypeScript.

11 月 25 日的规范发布增加了异步操作、无状态性、服务器身份和官方扩展。主要编程语言的 MCP SDK 在 Python 和 TypeScript 上已实现每月 9700 万次以上的下载。

## The Agentic AI Foundation
## 代理式 AI 基金会

The AAIF operates as a directed fund under the Linux Foundation, co-founded by Anthropic, Block, and OpenAI, with support from Google, Microsoft, AWS, Cloudflare, and Bloomberg.

AAIF 作为 Linux 基金会下的定向基金运营，由 Anthropic、Block 和 OpenAI 联合创立，并得到 Google、Microsoft、AWS、Cloudflare 和 Bloomberg 的支持。

MCP joins "goose" (Block) and "AGENTS.md" (OpenAI) as founding projects. The foundation aims to advance agentic AI development transparently and in the public interest through strategic investment and open standards development.

MCP 与 "goose"（Block）和 "AGENTS.md"（OpenAI）一起成为创始项目。该基金会旨在通过战略投资和开放标准开发，透明地推进代理式 AI 的发展，服务于公共利益。

MCP's governance structure remains unchanged, with maintainers continuing to prioritize community input and transparent decision-making.

MCP 的治理结构保持不变，维护者继续优先考虑社区意见和透明的决策。
