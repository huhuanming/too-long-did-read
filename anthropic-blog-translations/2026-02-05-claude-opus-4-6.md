# Introducing Claude Opus 4.6
# 介绍 Claude Opus 4.6

> Source: https://www.anthropic.com/news/claude-opus-4-6
> Date: 2026-02-05

---

Anthropic unveiled Claude Opus 4.6, representing significant improvements in coding capabilities, agentic task performance, and long-context processing. The model introduces a 1M token context window in beta—a first for Anthropic's Opus-class models.

Anthropic 发布了 Claude Opus 4.6，在编程能力、代理任务表现和长上下文处理方面实现了显著改进。该模型引入了处于测试阶段的 100 万 token 上下文窗口——这是 Anthropic Opus 级别模型的首次。

---

## Key Improvements
## 主要改进

The new model enhances several core competencies. It "plans more carefully, sustains agentic tasks for longer, can operate more reliably in larger codebases," and demonstrates superior code review and debugging abilities. Notably, Opus 4.6 achieves the highest score on Terminal-Bench 2.0 for agentic coding and leads frontier models on Humanity's Last Exam, a multidisciplinary reasoning assessment.

新模型增强了多项核心能力。它"规划更加仔细，能更长时间地维持代理任务，在更大的代码库中运行更可靠"，并展现出卓越的代码审查和调试能力。值得注意的是，Opus 4.6 在 Terminal-Bench 2.0 代理编程测试中取得了最高分，并在多学科推理评估 Humanity's Last Exam 中领先所有前沿模型。

On GDPval-AA—evaluating economically valuable knowledge work—Opus 4.6 outperforms GPT-5.2 by approximately 144 Elo points and its predecessor, Claude Opus 4.5, by 190 points.

在评估具有经济价值的知识工作的 GDPval-AA 测试中，Opus 4.6 以约 144 Elo 分的优势超越 GPT-5.2，以 190 分的优势超越其前代 Claude Opus 4.5。

---

## Technical Features
## 技术特性

New API capabilities include adaptive thinking, enabling the model to determine when extended reasoning proves beneficial; configurable effort levels controlling intelligence, speed, and cost; and context compaction for longer-running tasks. The model supports 128k output tokens.

新的 API 功能包括自适应思维，使模型能够判断何时需要扩展推理；可配置的努力级别以控制智能度、速度和成本；以及用于长时间运行任务的上下文压缩。该模型支持 128k 输出 token。

---

## Product Enhancements
## 产品增强

Claude Code now features agent teams for parallel work coordination. Claude in Excel received substantial upgrades, while Claude in PowerPoint launches in research preview, enabling document structuring and visual presentation generation.

Claude Code 现在具有用于并行工作协调的代理团队功能。Claude in Excel 获得了重大升级，同时 Claude in PowerPoint 以研究预览的形式推出，支持文档结构化和视觉演示文稿的生成。

---

## Safety Profile
## 安全概况

Testing demonstrates an "overall safety profile as good as, or better than, any other frontier model," with low misaligned behavior rates and minimal over-refusals on benign queries.

测试表明其"整体安全概况与任何其他前沿模型一样好，甚至更好"，错误对齐行为率低，对良性查询的过度拒绝也很少。

---

## Pricing and Availability
## 定价与可用性

Opus 4.6 is available on Claude.ai, via API using `claude-opus-4-6`, and through major cloud platforms. Standard pricing remains $5/$25 per million tokens, with premium pricing ($10/$37.50) for prompts exceeding 200k tokens.

Opus 4.6 可在 Claude.ai 上使用，通过 API 使用 `claude-opus-4-6`，以及通过主要云平台使用。标准定价保持每百万 token 5 美元/25 美元，超过 200k token 的提示词采用高级定价（10 美元/37.50 美元）。
