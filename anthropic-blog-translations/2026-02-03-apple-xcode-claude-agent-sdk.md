# Apple's Xcode Now Supports the Claude Agent SDK
# Apple Xcode 现已支持 Claude Agent SDK

> Source: https://www.anthropic.com/news/apple-xcode-claude-agent-sdk
> Date: 2026-02-03

---

Xcode 26.3 introduces native integration with the Claude Agent SDK, enabling developers to leverage autonomous AI capabilities directly within Apple's development environment for iOS, iPad, Mac, Apple Watch, Apple Vision Pro, and Apple TV.

Xcode 26.3 引入了与 Claude Agent SDK 的原生集成，使开发者能够直接在 Apple 的开发环境中利用自主 AI 能力，支持 iOS、iPad、Mac、Apple Watch、Apple Vision Pro 和 Apple TV 开发。

---

## Visual Verification with Previews
## 通过预览进行视觉验证

Claude can now capture Xcode Previews to observe interface implementations, identify visual issues, and iterate independently. This proves especially valuable for SwiftUI development, where visual output is paramount.

Claude 现在可以捕获 Xcode 预览来观察界面实现、识别视觉问题并独立进行迭代。这对于 SwiftUI 开发尤其有价值，因为视觉输出至关重要。

---

## Reasoning Across Projects
## 跨项目推理

The AI examines complete project file structures, understanding how frameworks like SwiftUI, UIKit, and Swift Data interconnect. Rather than focusing on individual files, Claude comprehends the entire application architecture before implementing changes.

AI 检查完整的项目文件结构，理解 SwiftUI、UIKit 和 Swift Data 等框架如何相互连接。Claude 不是专注于单个文件，而是在实施更改之前理解整个应用程序架构。

---

## Autonomous Task Execution
## 自主任务执行

Developers can now provide high-level goals rather than step-by-step instructions. Claude breaks down tasks independently, decides which files require modification, executes changes, and continues iteration until completion or user input becomes necessary.

开发者现在可以提供高层级的目标，而非逐步指令。Claude 独立分解任务，决定哪些文件需要修改，执行更改，并持续迭代直到完成或需要用户输入。

---

## Model Context Protocol Integration
## 模型上下文协议集成

Beyond IDE integration, Xcode 26.3 makes capabilities accessible via the Model Context Protocol, allowing Claude Code users to integrate with Xcode over MCP and capture visual previews from the command line.

除了 IDE 集成外，Xcode 26.3 还通过模型上下文协议提供了更多可访问的能力，允许 Claude Code 用户通过 MCP 与 Xcode 集成，并从命令行捕获视觉预览。

---

## Availability
## 可用性

Xcode 26.3 is available as a release candidate for Apple Developer Program members, with a general App Store release forthcoming.

Xcode 26.3 已作为候选发布版本提供给 Apple Developer Program 会员，即将在 App Store 正式发布。
