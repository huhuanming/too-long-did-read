# ServiceNow Chooses Claude to Power Customer Apps and Increase Internal Productivity
# ServiceNow 选择 Claude 驱动客户应用并提高内部生产力

> Source: https://www.anthropic.com/news/servicenow-anthropic-claude
> Date: 2026-01-28

---

ServiceNow has selected Claude as its default model for ServiceNow Build Agent and as a preferred model across the ServiceNow AI Platform. The partnership aims to help enterprises integrate AI into core business operations with attention to scale and security.

ServiceNow 选择 Claude 作为 ServiceNow Build Agent 的默认模型，以及 ServiceNow AI 平台上的首选模型。该合作旨在帮助企业将 AI 整合到核心业务运营中，同时关注规模和安全性。

---

## For ServiceNow Customers
## 面向 ServiceNow 客户

Claude powers ServiceNow Build Agent, enabling developers of all skill levels to build applications using natural language prompts. The platform expects usage to quadruple within 12 months. ServiceNow targets a 50% reduction in implementation time for customers through Claude integration.

Claude 驱动 ServiceNow Build Agent，使各技能水平的开发者都能使用自然语言提示构建应用。该平台预计使用量将在 12 个月内翻两番。ServiceNow 的目标是通过 Claude 集成将客户的实施时间缩短 50%。

The partnership includes industry-specific applications for healthcare and life sciences, where Claude will support research analysis and claims authorization—potentially reducing processing from days to hours.

该合作包括面向医疗保健和生命科学的行业特定应用，其中 Claude 将支持研究分析和理赔授权——有可能将处理时间从数天缩短到数小时。

---

## Internal Implementation
## 内部实施

ServiceNow is deploying Claude across its 29,000-person workforce. Sales teams using a Claude-powered coaching tool saw preparation time cut by 95%. Engineering teams utilize Claude Code to write, review, and debug code while automating repetitive development tasks.

ServiceNow 正在其 29,000 人的团队中部署 Claude。使用 Claude 驱动的辅导工具的销售团队将准备时间缩短了 95%。工程团队利用 Claude Code 编写、审查和调试代码，同时自动化重复性的开发任务。

## Executive Perspective
## 高管观点

Dario Amodei, Anthropic CEO, emphasized that "the way to get much better results is to make AI an integral part of how you get work done—woven into the whole range of things workers do every day."

Anthropic 首席执行官 Dario Amodei 强调，"获得更好结果的方式是让 AI 成为你完成工作方式的不可或缺的部分——融入员工每天所做的各种事情中。"

## Availability
## 可用性

Claude is now available as the default model for Build Agent and preferred model across ServiceNow's AI Platform. Tens of thousands of enterprise customers can access these capabilities at servicenow.com/ai.

Claude 现已作为 Build Agent 的默认模型和 ServiceNow AI 平台上的首选模型提供。数万名企业客户可以在 servicenow.com/ai 上使用这些功能。
