# Introducing Claude Opus 4.5
# 介绍 Claude Opus 4.5

> Source: https://www.anthropic.com/news/claude-opus-4-5
> Date: 2025-11-24

---

Anthropic has released Claude Opus 4.5, described as "intelligent, efficient, and the best model in the world for coding, agents, and computer use." The model demonstrates substantial improvements across everyday tasks including deep research and spreadsheet manipulation.

Anthropic 发布了 Claude Opus 4.5，被描述为"智能、高效，是全球编程、代理和计算机使用方面最好的模型。"该模型在日常任务中展示了重大改进，包括深度研究和电子表格操作。

---

## Key Performance Metrics
## 关键性能指标

Claude Opus 4.5 achieves state-of-the-art results on software engineering benchmarks. The model outperforms competitors on SWE-bench Verified and shows approximately 10.6% improvement over Sonnet 4.5 on coding problem-solving tasks.

Claude Opus 4.5 在软件工程基准测试中取得了最先进的结果。该模型在 SWE-bench Verified 上优于竞争对手，在编程问题解决任务上比 Sonnet 4.5 提高了约 10.6%。

---

## Availability and Pricing
## 可用性和定价

The model is accessible via Claude's apps, API, and major cloud platforms. Developers can access it using the identifier `claude-opus-4-5-20251101`. Pricing has been reduced to "$5/$25 per million tokens," making advanced capabilities more broadly available.

该模型可通过 Claude 的应用、API 和主要云平台访问。开发者可以使用标识符 `claude-opus-4-5-20251101` 访问。定价已降至"每百万 token 5 美元/25 美元"，使高级能力更广泛地可用。

---

## Technical Capabilities
## 技术能力

According to customer feedback, the model "handles ambiguity and reasons about tradeoffs without hand-holding." Users report that "tasks that were near-impossible for Sonnet 4.5 just a few weeks ago are now within reach."

根据客户反馈，该模型"在无需手把手指导的情况下处理模糊性并推理权衡。"用户报告说，"几周前 Sonnet 4.5 几乎不可能完成的任务现在已经触手可及。"

The model demonstrates enhanced performance in multi-step reasoning, tool use, and sustained autonomous reasoning tasks. One benchmark scenario showed the model finding creative legitimate solutions within policy constraints—behavior described as reflecting genuine problem-solving capability.

该模型在多步骤推理、工具使用和持续自主推理任务方面表现出增强的性能。一个基准测试场景显示，该模型在政策约束内找到了创造性的合法解决方案——这种行为被描述为反映了真正的问题解决能力。

---

## Safety Improvements
## 安全改进

Anthropic reports that Opus 4.5 is "the most robustly aligned model we have released to date." The model shows improved resistance to prompt injection attacks compared to other frontier models.

Anthropic 报告称 Opus 4.5 是"我们迄今为止发布的最稳健对齐的模型。"与其他前沿模型相比，该模型显示出对提示注入攻击的更强抵抗力。

---

## Product Enhancements
## 产品增强

Updates include Claude Code with improved planning capabilities, extended conversation support in Claude apps, and expanded integration with Excel, Chrome, and desktop applications.

更新包括具有改进规划能力的 Claude Code、Claude 应用中的扩展对话支持，以及与 Excel、Chrome 和桌面应用程序的扩展集成。
