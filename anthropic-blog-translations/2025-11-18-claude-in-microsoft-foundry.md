# Claude Now Available in Microsoft Foundry and Microsoft 365 Copilot
# Claude 现已在 Microsoft Foundry 和 Microsoft 365 Copilot 中可用

> Source: https://www.anthropic.com/news/claude-in-microsoft-foundry
> Date: 2025-11-18

---

Microsoft and Anthropic announced an expanded partnership making Claude models available in Microsoft Foundry. "Claude Sonnet 4.5, Haiku 4.5, and Opus 4.1 models are now available in public preview in Microsoft Foundry" where Azure customers can develop production applications.

微软和 Anthropic 宣布扩大合作伙伴关系，使 Claude 模型在 Microsoft Foundry 中可用。"Claude Sonnet 4.5、Haiku 4.5 和 Opus 4.1 模型现已在 Microsoft Foundry 中公开预览"，Azure 客户可以在其中开发生产应用程序。

The integration allows organizations to leverage Claude's capabilities while maintaining their existing Microsoft ecosystem. Developers can access Claude through Foundry's APIs and tools, with support for Python, TypeScript, and C# SDKs using Microsoft Entra authentication.

该集成允许组织在维护其现有微软生态系统的同时利用 Claude 的能力。开发者可以通过 Foundry 的 API 和工具访问 Claude，支持使用 Microsoft Entra 认证的 Python、TypeScript 和 C# SDK。

---

## Model Options
## 模型选项

**Sonnet 4.5** stands as the premier coding model, suited for sophisticated reasoning and autonomous coding tasks requiring state-of-the-art performance.

**Sonnet 4.5** 是首选的编程模型，适用于需要最先进性能的复杂推理和自主编程任务。

**Haiku 4.5** delivers near-frontier capabilities at reduced cost—approximately one-third of Sonnet's pricing—making it ideal for high-volume applications like customer support automation and content moderation.

**Haiku 4.5** 以更低的成本——约为 Sonnet 定价的三分之一——提供接近前沿的能力，适合客户支持自动化和内容审核等高流量应用。

**Opus 4.1** excels at complex, multi-step reasoning problems requiring sustained analytical focus.

**Opus 4.1** 擅长需要持续分析关注的复杂多步骤推理问题。

---

## Key Features
## 主要功能

All three models support various Claude Developer Platform capabilities including code execution, web search and fetch functionality, citations, vision processing, tool use, and prompt caching.

所有三个模型都支持各种 Claude 开发者平台功能，包括代码执行、网络搜索和获取功能、引用、视觉处理、工具使用和提示缓存。

---

## Deployment Details
## 部署详情

Claude is available in Global Standard deployment with standard API pricing. The US DataZone launches soon. Importantly, Claude qualifies for Microsoft Azure Consumption Commitment eligibility, streamlining procurement for existing Microsoft customers.

Claude 在全球标准部署中以标准 API 定价提供。美国数据区即将推出。重要的是，Claude 符合 Microsoft Azure 消费承诺的资格，简化了现有微软客户的采购流程。

Microsoft 365 Copilot integration includes Claude powering the Researcher agent and enabling custom agent development through Copilot Studio. Excel's Agent Mode now offers Claude integration for spreadsheet building and editing tasks.

Microsoft 365 Copilot 集成包括 Claude 驱动 Researcher 代理，并通过 Copilot Studio 实现自定义代理开发。Excel 的 Agent Mode 现在提供 Claude 集成，用于电子表格构建和编辑任务。
