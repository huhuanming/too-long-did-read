# Claude Code on the Web
# 网页版 Claude Code

> Source: https://www.anthropic.com/news/claude-code-on-the-web
> Date: 2025-11-12

---

Anthropic introduced Claude Code on the web, enabling developers to "assign multiple coding tasks to Claude that run on Anthropic-managed cloud infrastructure." This beta research preview allows parallel task execution across repositories directly from a browser interface.

Anthropic 推出了网页版 Claude Code，使开发者能够"将多个编程任务分配给在 Anthropic 管理的云基础设施上运行的 Claude。"这个测试版研究预览允许直接从浏览器界面跨多个代码仓库并行执行任务。

---

## Core Features
## 核心功能

The platform offers several capabilities:

该平台提供多项功能：

- **Parallel Execution:** Users can run multiple coding sessions simultaneously across different repositories from a single interface
- **并行执行：** 用户可以从单一界面在不同的代码仓库中同时运行多个编程会话

- **GitHub Integration:** Connect repositories and describe needed work for Claude to implement
- **GitHub 集成：** 连接代码仓库并描述需要 Claude 实施的工作

- **Real-time Tracking:** Monitor progress as Claude works through tasks
- **实时追踪：** 在 Claude 处理任务时监控进度

- **Auto PR Creation:** Automatic pull request generation with change summaries
- **自动创建 PR：** 自动生成拉取请求并附带变更摘要

- **Mobile Support:** Available on iOS app for on-the-go development
- **移动支持：** 在 iOS 应用上可用，支持移动开发

---

## Ideal Use Cases
## 理想使用场景

The tool works effectively for "answering questions about how projects work," bugfixes, routine tasks, and backend changes using test-driven development practices.

该工具在"回答关于项目如何工作的问题"、修复 bug、日常任务以及使用测试驱动开发实践的后端更改方面表现出色。

---

## Security Implementation
## 安全实施

Claude Code runs in isolated sandbox environments with "network and filesystem restrictions." Git interactions use a secure proxy service, and users can configure custom network access to control which domains Claude can connect to.

Claude Code 运行在带有"网络和文件系统限制"的隔离沙箱环境中。Git 交互使用安全代理服务，用户可以配置自定义网络访问以控制 Claude 可以连接到哪些域。

---

## Availability
## 可用性

The research preview launched for Pro and Max users, with expansion to Team and Enterprise premium seats. Access is available at claude.com/code.

研究预览版面向 Pro 和 Max 用户推出，并将扩展到团队版和企业版高级席位。可在 claude.com/code 上使用。
