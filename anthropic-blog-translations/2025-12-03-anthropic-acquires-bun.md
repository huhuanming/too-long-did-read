# Anthropic Acquires Bun as Claude Code Reaches $1B Milestone
# Anthropic 收购 Bun，Claude Code 达到 10 亿美元里程碑

> Source: https://www.anthropic.com/news/anthropic-acquires-bun-as-claude-code-reaches-usd1b-milestone
> Date: 2025-12-03

---

Anthropic announced the acquisition of Bun, a JavaScript runtime, following Claude Code's achievement of $1 billion in run-rate revenue just six months after its public launch in May 2025.

Anthropic 宣布收购 JavaScript 运行时 Bun，此前 Claude Code 在 2025 年 5 月公开发布仅六个月后就实现了 10 亿美元的年化收入。

---

## Claude Code's Growth
## Claude Code 的增长

Claude Code has become integral to leading enterprises including Netflix, Spotify, KPMG, L'Oreal, and Salesforce. The platform represents "a new era of agentic coding, fundamentally changing how teams build software."

Claude Code 已成为 Netflix、Spotify、KPMG、欧莱雅和 Salesforce 等领先企业不可或缺的一部分。该平台代表着"代理式编程的新时代，从根本上改变了团队构建软件的方式。"

---

## About Bun
## 关于 Bun

Bun, founded by Jarred Sumner in 2021, serves as an all-in-one toolkit combining runtime, package manager, bundler, and test runner. The platform boasts over 7 million monthly downloads and 82,000+ GitHub stars, adopted by companies like Midjourney and Lovable.

Bun 由 Jarred Sumner 于 2021 年创立，是一个集运行时、包管理器、打包工具和测试运行器于一体的全功能工具包。该平台拥有每月超过 700 万次下载和 82,000 多个 GitHub stars，被 Midjourney 和 Lovable 等公司采用。

---

## Strategic Vision
## 战略愿景

Mike Krieger, Anthropic's Chief Product Officer, stated the acquisition aligns with the company's pursuit of "opportunities that bolster our technical excellence" while reinforcing leadership in enterprise AI.

Anthropic 首席产品官 Mike Krieger 表示，此次收购符合公司追求"增强我们技术卓越性的机会"的目标，同时巩固了在企业 AI 领域的领导地位。

---

## Commitment to Open Source
## 对开源的承诺

Bun will "remain open source and MIT-licensed," with Anthropic committed to advancing it as the preferred JavaScript and TypeScript development environment.

Bun 将"保持开源和 MIT 许可证"，Anthropic 承诺将其推进为首选的 JavaScript 和 TypeScript 开发环境。

---

The acquisition represents Anthropic's disciplined approach to acquisitions—pursuing opportunities that strengthen technical capabilities and align with company principles while supporting infrastructure for next-generation software development.

此次收购体现了 Anthropic 严谨的收购策略——追求能够增强技术能力、符合公司原则，同时支持下一代软件开发基础设施的机会。
