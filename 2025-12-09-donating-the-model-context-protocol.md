# 捐赠模型上下文协议并建立智能体 AI 基金会
# Donating the Model Context Protocol and Establishing the Agentic AI Foundation

> 原文链接: https://www.anthropic.com/news/donating-the-model-context-protocol-and-establishing-of-the-agentic-ai-foundation
> 日期: 2025-12-09

---

## English

Anthropic announced the donation of the Model Context Protocol (MCP) to the Linux Foundation's newly established Agentic AI Foundation (AAIF). This directed fund operates under the Linux Foundation and was co-founded by Anthropic, Block, and OpenAI, with additional support from Google, Microsoft, AWS, Cloudflare, and Bloomberg.

One year after its introduction, MCP has achieved substantial adoption across the AI ecosystem. The protocol now has more than 10,000 active public MCP servers deployed across various platforms including ChatGPT, Cursor, Gemini, Microsoft Copilot, and Visual Studio Code. Enterprise infrastructure providers including AWS, Cloudflare, Google Cloud, and Microsoft Azure now offer deployment support.

Claude currently features a directory with over 75 connectors powered by MCP. Recent API improvements include Tool Search and Programmatic Tool Calling capabilities designed to optimize production deployments handling thousands of tools while reducing latency in complex agent workflows.

The MCP Registry provides a community-driven discovery platform for available servers. The November 25th specification release introduced asynchronous operations, statelessness, server identity, and official extensions. Official SDKs exist for all major programming languages, with 97 million monthly downloads across Python and TypeScript.

The Linux Foundation brings decades of experience stewarding critical open-source projects including the Linux Kernel, Kubernetes, Node.js, and PyTorch. Under the AAIF, MCP will join goose (by Block) and AGENTS.md (by OpenAI) as founding projects. The protocol's governance model remains unchanged, with maintainers continuing to prioritize community input and transparent decision-making.

Anthropic emphasized that open-source software remains essential for building secure, innovative ecosystems for agentic AI, and commits to ongoing contributions through the AAIF.

## 中文翻译

Anthropic 宣布将模型上下文协议 (MCP) 捐赠给 Linux 基金会新成立的智能体 AI 基金会 (AAIF)。该定向基金在 Linux 基金会下运作，由 Anthropic、Block 和 OpenAI 共同创立，并得到 Google、Microsoft、AWS、Cloudflare 和 Bloomberg 的额外支持。

在推出一年后，MCP 在 AI 生态系统中实现了大规模采用。该协议现在有超过 10,000 个活跃的公共 MCP 服务器部署在各种平台上，包括 ChatGPT、Cursor、Gemini、Microsoft Copilot 和 Visual Studio Code。包括 AWS、Cloudflare、Google Cloud 和 Microsoft Azure 在内的企业基础设施提供商现在提供部署支持。

Claude 目前拥有一个由 MCP 驱动的超过 75 个连接器的目录。最近的 API 改进包括工具搜索和编程式工具调用功能，旨在优化处理数千个工具的生产部署，同时减少复杂代理工作流中的延迟。

MCP 注册表提供了一个社区驱动的可用服务器发现平台。11 月 25 日的规范发布引入了异步操作、无状态性、服务器身份和官方扩展。所有主要编程语言都有官方 SDK，Python 和 TypeScript 的月下载量达到 9700 万。

Linux 基金会带来了数十年管理关键开源项目的经验，包括 Linux 内核、Kubernetes、Node.js 和 PyTorch。在 AAIF 下，MCP 将与 goose（由 Block 提供）和 AGENTS.md（由 OpenAI 提供）一起成为创始项目。该协议的治理模式保持不变，维护者继续优先考虑社区输入和透明决策。

Anthropic 强调，开源软件对于构建安全、创新的智能体 AI 生态系统仍然至关重要，并承诺通过 AAIF 持续贡献。
